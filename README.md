# معالج الصوت الذكي 🎵

أداة احترافية لمعالجة الملفات الصوتية في المتصفح — بدون رفع ملفاتك لأي خادم.

## الميزات

- ✂️ **التقطيع اليدوي** — حدد وقص أي جزء من الملف
- 🔇 **إزالة الصمت** — اكتشاف وحذف فترات الصمت تلقائياً
- 🧩 **دمج الملفات** — اجمع عدة ملفات صوتية في ملف واحد
- ⚡ **تحسين الصوت** — تقليل الضوضاء وتحسين الوضوح
- 🕌 **وضع الصلاة** — معالجة تلقائية مخصصة لتسجيلات الصلاة
- 📝 **استخراج النص** — تحويل الصوت إلى نص (يتطلب OpenAI API Key)

## الخصوصية

جميع المعالجة تتم في متصفحك مباشرةً — لا يُرفع أي ملف صوتي لأي خادم.

## الصيغ المدعومة

MP3 · WAV · M4A · OGG · WebM — حتى 100 MB

## التشغيل المحلي

```bash
# تثبيت المتطلبات
pnpm install

# تشغيل بيئة التطوير
pnpm dev

# فتح المتصفح على
http://localhost:3000
```

## متغيرات البيئة (اختيارية)

```env
# لتفعيل ميزة استخراج النص
OPENAI_API_KEY=sk-...

# لتقييد الوصول لنطاقات محددة
ALLOWED_ORIGINS=https://your-domain.com
```

## النشر

المشروع جاهز للنشر على Vercel بضغطة واحدة.

## التقنيات

- React 19 + TypeScript + Vite
- Web Audio API + Web Workers
- WaveSurfer.js
- Tailwind CSS + shadcn/ui
